# Website Áo Dài Việt

Website giới thiệu về áo dài Việt Nam và dịch vụ cho thuê áo dài.

## Giới thiệu

Đây là website giới thiệu về áo dài truyền thống Việt Nam, bao gồm:
- Lịch sử và nguồn gốc áo dài
- Các loại áo dài phổ biến
- Ý nghĩa văn hóa
- Dịch vụ cho thuê áo dài và thuê thợ chụp ảnh

## Cách sử dụng

1. Mở file `index.html` bằng trình duyệt
2. Thêm ảnh vào thư mục `images/` nếu cần
3. Chỉnh sửa nội dung trong file HTML

**Lưu ý:** Trên mobile, bấm vào logo để mở menu

## Tính năng chính

- Menu điều hướng với dropdown
- Thanh tìm kiếm
- Thư viện ảnh
- Dịch vụ cho thuê áo dài (4 mẫu)
- Dịch vụ thuê thợ chụp ảnh
- Responsive trên mọi thiết bị

## Công nghệ

- HTML5
- CSS3
- JavaScript
- Google Fonts
